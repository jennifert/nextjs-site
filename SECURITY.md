# Security Policy

## Supported Versions

This project is actively maintained on the `main` branch.

| Version | Supported |
| ------- | --------- |
| main    | ✅ |

## Reporting a Vulnerability

If you discover a security issue, please use GitHub’s private vulnerability reporting feature for this repository rather than opening a public issue.

You can do that from the repository’s Security tab by choosing **Report a vulnerability**.

Please include:

- a description of the issue
- steps to reproduce it
- any relevant screenshots or proof of concept
- the affected page, route, or dependency if known

I will review the report and address it as time allows.

## Notes

This site is a personal project and does not currently include user accounts, payments, or private user data handling. However, security issues related to dependencies, deployment, or site behavior are still appreciated.
