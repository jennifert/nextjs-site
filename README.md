# nextjs-site
The current website for jenntesolin.com. Built with NextJS and TailWind CSS.

This is still under construction so please excuse any weird styles.

## Requirements
- Node: latest version
- NPM 7.x

- Tailwind: 2.x
- PostCSS: 8.x
- Next: 12.x
- Next PWA: 5.x
- PrismJS: 1.x
- React & React-Dom: 17.x
- ESLint: 7.x
