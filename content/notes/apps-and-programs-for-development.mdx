Below is an updated list of applications and services you can use for web development. This time, I grouped them into free, paid, and subscription-based tools.

## Cross-platform apps

### Free

- <a href="https://www.sourcetreeapp.com/" target="_blank" rel="nofollow noopener noreferrer">SourceTree</a> - a desktop app for committing and managing source code
- <a href="https://code.visualstudio.com/" target="_blank" rel="nofollow noopener noreferrer">Visual Studio Code</a> - a powerful text editor
- <a href="https://netbeans.apache.org/download/index.html" target="_blank" rel="nofollow noopener noreferrer">NetBeans</a> - an IDE primarily used for Java, but also useful for PHP, HTML, and other web development work. You can add extensions for many different workflows.
- <a href="https://www.virtualbox.org/" target="_blank" rel="nofollow noopener noreferrer">VirtualBox</a> - a virtual machine manager

### Subscription

- <a href="https://www.adobe.com/creativecloud.html" target="_blank" rel="nofollow noopener noreferrer">Adobe Creative Cloud</a> - tools for working with graphics, PDFs, video, and more
- <a href="https://www.jetbrains.com/phpstorm/" target="_blank" rel="nofollow noopener noreferrer">PhpStorm</a> - an IDE for PHP development
- <a href="https://www.jetbrains.com/webstorm/" target="_blank" rel="nofollow noopener noreferrer">WebStorm</a> - an IDE for web development
- <a href="https://nordvpn.com/" target="_blank" rel="nofollow noopener noreferrer">NordVPN</a> - a VPN service provider

### Paid

- <a href="https://cyberduck.io/" target="_blank" rel="nofollow noopener noreferrer">Cyberduck</a> - a file transfer application
- <a href="http://www.sublimetext.com/" target="_blank" rel="nofollow noopener noreferrer">Sublime Text</a> - a text editor

## Windows-specific

### Free

- <a href="http://notepad-plus-plus.org/" target="_blank" rel="nofollow noopener noreferrer">Notepad++</a> - a lightweight text editor
- <a href="http://winscp.net/eng/index.php" target="_blank" rel="nofollow noopener noreferrer">WinSCP</a> - a file transfer application

## Mac-specific

### Paid

- <a href="https://www.sketch.com/" target="_blank" rel="nofollow noopener noreferrer">Sketch</a> - a design toolkit
